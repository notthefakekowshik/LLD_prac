# LLD Preparation Roadmap

This roadmap is designed to take you from writing "just-working" code to building extensible, enterprise-grade systems. Complete one phase before moving to the next.

## Phase 1: The Foundations (The Grammar)
- [ ] **S.O.L.I.D Principles** Mastery
  - Implement 5 scenarios violating each principle and refactor them.
- [ ] **Object Oriented Programming** (Deep Dive)
  - Composition vs Inheritance.
  - Abstract Classes vs Interfaces.
- [ ] **UML Mastery**
  - Drawing Class and Sequence diagrams using Mermaid.js.

## Phase 2: Design Pattern Catalog (The Vocabulary)
- [ ] **Creational Patterns**
  - Singleton, Factory, Abstract Factory, Builder, Prototype.
- [ ] **Structural Patterns**
  - Adapter, Bridge, Decorator, Facade, Flyweight, Proxy.
- [ ] **Behavioral Patterns**
  - Strategy, Observer, State, Command, Template Method, Iterator.

## Phase 3: Building Blocks (Reusable Components)
- [ ] **In-Memory Cache** (with eviction policies).
- [ ] **Custom Thread Pool** (Executors implementation).
- [ ] **Rate Limiter** (Token Bucket, Leaky Bucket).
- [ ] **Logging Framework** (Logger, Handlers, Formatters).
- [ ] **Task Scheduler** (Cron-like scheduler).

## Phase 4: Masterpieces (Machine Coding Problems)
- [ ] **Parking Lot System** (Multiple Levels, Multi-vehicle, EV charging).
- [ ] **Movie Booking System** (BookMyShow).
- [ ] **Splitwise** (Expense sharing with multiple algorithms).
- [ ] **Snake and Ladder / Chess**.
- [ ] **Vending Machine**.
- [ ] **Atm Machine**.
- [ ] **Hotel Management System**.
- [ ] **Library Management System**.
