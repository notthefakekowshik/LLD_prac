## Quick Reference

- **LinkedHashMap**: Has TWO structures - hash table (O(1) get) + doubly linked list (order). The DLL is GLOBAL across all buckets, not per-bucket.
- **HashSet**: Facade over HashMap where keys = set elements, values = shared dummy Object (PRESENT).
- **LinkedHashSet**: Facade over LinkedHashMap with insertion order iteration.
